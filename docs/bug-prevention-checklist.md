# Bug Prevention Checklist

- [ ] PDF 업로드 후 문제 수가 1개 이상인지 확인
- [ ] `순차 1문제` 모드에서 제출 후 다음 문제로 이동하는지 확인
- [ ] `랜덤 5문제` 모드에서 문제 순서가 고정되지 않는지 확인
- [ ] 퀴즈 완료 후 오답이 오답노트에 저장되는지 확인
- [ ] `오답만 풀기` 클릭 시 오답 문항만 재출제되는지 확인
- [ ] 로컬 스토리지 데이터가 새로고침 후 유지되는지 확인
- [ ] `정답 확인`만 눌렀을 때는 진행도(문항 수/%)가 증가하지 않는지 확인
- [ ] `정답 확인` 후 `다음 문제` 클릭 시에만 진행도가 증가하는지 확인
- [ ] 문제풀이 상단의 정답/오답 카운터가 판정 결과와 일치하는지 확인
- [ ] 보기/문항에 저작권 문구/페이지 번호/머리말이 섞이지 않는지 확인
- [ ] 문제집 삭제 버튼 클릭 시 확인창이 먼저 노출되는지 확인
- [ ] 문제집 삭제 확인 시 문제집/연결 오답 기록이 함께 삭제되는지 확인
- [ ] 문제집 삭제 취소 시 데이터가 유지되는지 확인
- [ ] 이미지 포함 문항에서 그래프/도표가 함께 표시되는지 확인
- [ ] 이미지 파싱 적용 후에도 업로드가 120초 내 완료되는지 확인
- [ ] 실제 PDF 업로드(사용자 파일 경로)로 문제 추출 성공하는지 확인
- [ ] `정보처리기사` 파일명(NFD/NFC 모두)에서 문항 수가 1~100으로 유지되는지 확인
- [ ] 파싱 결과에 101번 이상 문항 번호가 포함되지 않는지 확인
- [ ] 100문항 시험지에서 1~100 연속 번호 누락 없이 생성되는지 확인
- [ ] 업로드 실패 시 사용자 안내 메시지(`uploadNotice`)가 즉시 노출되는지 확인
- [ ] 업로드 성공 시 사용자 안내 메시지(`문제집이 추가되었습니다`)가 노출되는지 확인
- [ ] 업로드 실패 후에도 앱이 멈추지 않고 재업로드가 가능한지 확인
- [ ] `node scripts/full-flow-scan.mjs` 실행 후 전 화면 스크린샷이 생성되는지 확인
- [ ] `npm run test`, `npm run build`가 모두 통과하는지 확인
- [ ] `npm run quality:gate` 실행 결과가 성공이고 `ui-scan/quality-gate-report.json`이 생성되는지 확인
- [ ] Firebase 환경변수 미설정 상태에서 앱이 정상(로컬 모드) 동작하는지 확인
- [ ] 구글 로그인 후 클라우드 백업/복원이 성공하는지 확인
- [ ] JSON 내보내기/가져오기 후 문항 수/오답/책갈피 데이터가 유지되는지 확인
- [ ] 무료 보호모드 한도 도달 시 자동 동기화가 중지되는지 확인
- [ ] 무료 보호모드 도달 이후에도 로컬 학습/JSON 백업이 정상 동작하는지 확인
- [ ] `npm run scan:uploaded-pdfs` 결과에서 placeholder 대상 문항이 `hasFigure: true`인지 확인
- [ ] 이미지 의존 문항의 보기가 `이미지의 n번 보기` 형태로 표시되는지 확인
