# QDF 버튼 플로우 (처음부터 끝까지)

## 1) 보관함 화면
- **PDF 선택 영역 클릭**
  - 입력: `#pdf-file-input` 파일 선택
  - 기대 결과: 스캔 완료 후 보관함 목록에 문제집 카드가 추가되고 현재 화면 유지
  - 기대 결과: 성공 안내 메시지(`문제집이 추가되었습니다`)가 노출
  - 실패 조건: 120초 내 문제집 카드 미노출

- **업로드 실패 케이스**
  - 입력: 잘못된 파일 형식, 손상 PDF, 파싱 실패 PDF
  - 기대 결과: 실패 안내 메시지가 즉시 노출되고 앱이 멈추지 않음
  - 기대 결과: 사용자 재시도(파일 재선택) 가능

- **문제집 카드 클릭**
  - 입력: 문제집 카드(`.libraryBook`) 클릭
  - 기대 결과: 해당 문제집 선택 상태 활성

- **다음 버튼 클릭**
  - 입력: `다음`
  - 기대 결과: 선택된 문제집 기준으로 `출제 설정` 화면 이동

- **삭제 버튼 클릭**
  - 입력: 카드 우측 상단 `X`
  - 기대 결과: 삭제 확인창 노출
  - 확인 시: 선택한 문제집과 연결된 오답 기록 동시 삭제
  - 취소 시: 데이터 변경 없음

## 2) 출제 설정 화면
- **학습 타입 선택**
  - 입력: `문제 풀기` 또는 `오답풀기`
  - 기대 결과: 다음 단계(문제 수) 노출
  - 예외: 오답 0개면 `오답풀기` 비활성

- **문제 수 선택**
  - 입력: `1문제` 또는 `5문제`
  - 기대 결과: 다음 단계(출제 순서) 노출

- **출제 순서 선택**
  - 입력: `순차` 또는 `랜덤`
  - 기대 결과: `시작하기` 버튼 노출 및 활성

- **시작하기 버튼 클릭**
  - 기대 결과: `문제 풀이` 화면 진입

## 3) 문제 풀이 화면
- **보기 선택 -> 정답 확인**
  - 기대 결과: 정답/오답 배지 노출 + 자동으로 다음 문제로 넘어가지 않음
  - 기대 결과: 상단 정오답 카운터(정답/오답)가 현재 판정 상태를 반영

- **다음 문제 클릭**
  - 기대 결과: 다음 문항 이동 + 진행도 증가
  - 검증 포인트: `정답 확인`만 누른 상태에서는 진행도/백분율이 증가하지 않아야 함

- **마지막 문항 완료**
  - 기대 결과: 출제 설정 화면 복귀 + 오답 기록 갱신

- **문항 텍스트 품질**
  - 기대 결과: 저작권 안내, 페이지 번호, 문서 머리말/꼬리말 같은 불필요 문구가 보기/문항에 섞이지 않음

## 4) 오답풀이 플로우
- 오답이 1개 이상이면 `오답풀기` 선택 가능
- 오답이 0개면 비활성 상태 유지
